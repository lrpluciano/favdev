O que é um Algoritmo?

Definição: Uma sequência finita de passos definidos e organizados logicamente para resolver um problema ou realizar uma tarefa.

Pense em uma receita de bolo:

  • Ingredientes: São os seus dados de entrada.
  • Modo de Preparo: É a sequência de passos (o algoritmo).
  • Bolo Pronto: É o resultado final.

A Principal Característica: Sequência Lógica de Passos.

Usando Linguagem Natural

É simplesmente escrever os passos do algoritmo em uma linguagem comum, como o português.

Vantagem Principal: Facilidade de Entendimento Humano.

  • Qualquer pessoa pode ler e entender.
  • É ótima para o primeiro rascunho.

Exemplo: Como Fazer um Café

  1. Pegar um filtro de papel.
  2. Colocar o pó de café no filtro.
  3. Aquecer a água até ferver.
  4. Despejar a água quente sobre o pó.
  5. Servir.

Ilustrando a Lógica com Fluxogramas

É um desenho que usa símbolos padronizados para mostrar o fluxo do algoritmo.

Função Principal: Ilustrar Visualmente a Lógica.

Símbolos Mais Comuns:

  • (Oval) Terminais: Início e Fim.
  • (Retângulo) Processamento: Uma ação a ser executada.
  • (Losango) Decisão: Uma pergunta com duas respostas (Sim/Não).
  • (Paralelogramo) Entrada/Saída: Ler dados ou mostrar resultados.

A Ponte entre a Língua e o Código

É uma forma de escrever o algoritmo que se parece com um programa, mas sem as regras rígidas de uma linguagem específica.

Por que usamos? Para representar a lógica sem se preocupar com a sintaxe.

Exemplo: Verificar se um aluno foi aprovado

INÍCIO
  ESCREVA "Digite a nota do aluno:"
  LEIA nota
  
  SE nota >= 7 ENTÃO
    ESCREVA "Aluno Aprovado!"
  SENÃO
    ESCREVA "Aluno Reprovado."
  FIM_SE
FIM

Tipos de Algoritmos

Algoritmos podem ser classificados pelo tipo de tarefa que realizam.

Algoritmo de Decisão: Tomar decisões baseadas em condições.

  • Exemplo: Um sistema que decide se um cartão de crédito deve ser bloqueado.

Outros Tipos Comuns:

  • Ordenação: Organizam uma lista (ex: nomes em ordem alfabética).
  • Busca: Procuram por um item (ex: buscar um contato).

O que faz um Algoritmo ser Bom?

Um algoritmo não só precisa funcionar, mas ser bem construído.

Característica Principal: Clareza

  • Deve ser fácil de ler e entender.
  • Evite ambiguidade.

Outras Qualidades Essenciais:

  • Eficiência: Mínimo de recursos (tempo e memória).
  • Correção: Produz sempre o resultado esperado.
  • Finitude: Deve sempre terminar.

A Etapa Mais Importante: Análise do Problema

Antes de escrever qualquer código, você precisa entender profundamente o que precisa ser resolvido.

O Processo de Desenvolvimento:

  1. 1. Análise e Entendimento do Problema: O que precisa ser feito? (Esta etapa vem primeiro!)
  2. 2. Planejamento da Solução: Desenhar um fluxograma ou escrever um pseudocódigo.
  3. 3. Implementação: Escrever o código em uma linguagem de programação.
  4. 4. Testes e Otimização: Verificar se o programa funciona corretamente.
"Não se pode resolver um problema que não se entende."

O Papel da Linguagem de Programação

Um algoritmo é um plano abstrato. A linguagem de programação é a ferramenta que torna esse plano real para o computador.

Papel Principal: Traduzir algoritmos em instruções compreensíveis pela máquina.

  • O computador não entende português nem pseudocódigo.
  • A linguagem de programação (Python, Java, C++) fornece as regras e o vocabulário que o computador executa.

Em resumo:

  • Algoritmo: A lógica, a estratégia.
  • Linguagem de Programação: O meio para comunicar essa estratégia.

Onde os Algoritmos Estão?

Algoritmos estão por toda parte no nosso mundo digital, tornando nossas vidas mais fáceis e eficientes.

Aplicação Principal: Tomada de Decisão Automatizada.

Exemplos no seu dia a dia:

  • GPS (Google Maps, Waze): Usa algoritmos de busca e de "caminho mais curto".
  • Netflix e Spotify: Usam algoritmos para recomendar novos conteúdos.
  • Buscadores (Google): Usam algoritmos de ordenação para colocar os resultados mais relevantes no topo.
  • Bancos: Usam algoritmos para analisar seu perfil e decidir se concede um empréstimo.

Resumo: O que Aprendemos?

  • Algoritmo é uma receita lógica para resolver problemas.
  • Podemos descrevê-lo com Linguagem Natural, Fluxogramas (visual) e Pseudocódigo (estruturado).
  • Um bom algoritmo é claro, correto e eficiente.
  • O primeiro passo é analisar e entender o problema.
  • A Linguagem de Programação é a ferramenta que transforma o algoritmo em um programa funcional.
  • Eles são essenciais para a tomada de decisão automatizada.
