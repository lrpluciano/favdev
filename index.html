1. O que é PHP?

PHP é uma linguagem de programação muito usada para criar páginas web dinâmicas. Seu nome é um acrônimo recursivo: PHP: Hypertext Preprocessor.

Características principais:

  • É gratuita e de código aberto (open source)
  • Roda no servidor (não no navegador do usuário)
  • Pode ser embutida dentro do HTML
  • Funciona em vários sistemas: Windows, Linux, macOS, etc.
  • É fracamente tipada (não precisa declarar o tipo da variável)
  • Suporta Programação Orientada a Objetos (POO)

Por que PHP é importante? Quando você acessa um site como um e-commerce, o que aparece na tela (seu nome, carrinho, produtos) muitas vezes vem de um banco de dados. O PHP é quem "busca" essas informações e monta a página antes de enviar ao seu navegador. Por isso, ele é essencial no desenvolvimento web do lado do servidor.

3. Variáveis e Tipos de Dados

Em PHP, variáveis começam com $ e não precisam de declaração de tipo:

$nome = "Ana";
$idade = 25;
$altura = 1.65;
$ativo = true;

Tipos comuns: string, inteiro, float (número com vírgula) e booleano.

Escopo de variáveis:

  • Local: só existe dentro de uma função.
  • Global: acessível em todo o script (use global dentro de funções).
  • Estática: mantém seu valor entre chamadas de função (static).
function contador() {
  static $cont = 0;
  $cont++;
  echo $cont;
}
contador(); // 1
contador(); // 2

4. Saída de Dados

Use echo ou print para mostrar algo no navegador:

echo "Olá, mundo!";
print "
Bem-vindo!
";

Você pode misturar HTML e PHP livremente.

Formatação com printf()

printf("Preço: R$ %.2f", 19.9); // Mostra: Preço: R$ 19.90

Especificadores comuns: %s (string), %d (inteiro), %f (float).

5. Operadores

Aritméticos

  • +, -, *, /, % (módulo), ** (potência)

Comparação

  • == (igual valor)
  • === (igual valor e tipo)
  • != ou <> (diferente)
  • <, >, <=, >=

6. Estruturas de Controle

Condicional: if

$hora = date("H");
if ($hora < 12) {
  echo "Bom dia!";
} else {
  echo "Boa tarde!";
}

Repetição: for e while

for ($i = 1; $i <= 3; $i++) {
  echo "Contagem: $i
";
}

$i = 1;
while ($i <= 3) {
  echo "While: $i
";
  $i++;
}

7. Funções

Funções internas

  • phpinfo() – mostra configurações do PHP
  • mysqli_connect() – conecta a um banco de dados

Funções criadas pelo usuário

function saudacao($nome) {
  return "Olá, $nome!";
}
echo saudacao("Luciano"); // Olá, Luciano!

8. Manipulação de Arquivos

PHP pode ler, escrever e gerenciar arquivos no servidor.

Modos comuns de abertura:

  • r – leitura
  • w – escrita (apaga conteúdo anterior)
  • a – escrita no final do arquivo
$arq = fopen("dados.txt", "w");
fwrite($arq, "Texto salvo!");
fclose($arq);

Outras funções úteis:

  • fread() – lê conteúdo
  • file() – lê arquivo como array
  • copy(), unlink() – copiar ou apagar
  • $_FILES – usado em uploads de arquivos

9. Banco de Dados

PHP suporta vários bancos: MySQL, PostgreSQL, SQLite, MongoDB, etc.

Comandos SQL comuns:

  • SELECT – buscar dados
  • INSERT INTO – inserir
  • UPDATE – atualizar
  • DELETE – apagar
$conexao = mysqli_connect("localhost", "usuario", "senha", "banco");

Exemplo de consulta:

$resultado = mysqli_query($conexao, "SELECT nome FROM clientes WHERE cidade = 'São Paulo'");
while ($linha = mysqli_fetch_assoc($resultado)) {
  echo $linha['nome'] . "
";
}

10. Cookies

Cookies são pequenos dados salvos no navegador do usuário.

setcookie("usuario", "Luciano", time() + 3600); // Dura 1 hora

Úteis para lembrar preferências, login, etc.

11. Programação Orientada a Objetos (POO)

Permite organizar o código em "classes" e "objetos".

class Pessoa {
  public $nome;
  public function __construct($n) {
    $this->nome = $n;
  }
  public function apresentar() {
    return "Olá, eu sou $this->nome";
  }
}

$p = new Pessoa("Clara");
echo $p->apresentar();

Conceitos-chave:

  • Classe: modelo (ex: "Carro")
  • Objeto: instância da classe (ex: "meu carro vermelho")
  • Propriedades: atributos (cor, marca)
  • Métodos: ações (ligar, acelerar)
  • $this: refere-se ao próprio objeto dentro da classe
  • Visibilidade: public, protected, private
