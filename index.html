1. Introdução ao PHP

Seja bem-vindo ao mundo do PHP! Nesta aula, vamos desvendar uma das linguagens mais poderosas e populares para o desenvolvimento web. Vamos começar do zero?

O que é PHP?

PHP é um acrônimo para "PHP Hypertext Preprocessor". Pense nele como uma ferramenta que roda no servidor (o computador que hospeda seu site) para criar páginas web dinâmicas. Ao invés de um site estático, onde o conteúdo é sempre o mesmo, o PHP permite que o site mude, interaja com o usuário e se conecte a bancos de dados.

Dica: PHP é uma linguagem embutida no HTML. Isso significa que você pode misturar código PHP diretamente no seu arquivo HTML.

Características Principais

  • Linguagem Interpretada: O código é executado linha por linha, sem precisar de uma compilação prévia.
  • Fracamente Tipada: Você não precisa declarar o tipo de uma variável (se é texto, número, etc.). O PHP tenta adivinhar!
  • Multiplataforma: Roda em Windows, Linux, macOS e muitos outros sistemas.
  • Orientada a Objetos: Permite organizar o código de forma estruturada e reutilizável.

A Sintaxe Básica

Todo o código PHP deve estar dentro de suas tags especiais. A mais comum é <?php ... ?>. E não se esqueça: toda linha de código em PHP termina com um ponto e vírgula (;)!

PHP 
<?php
  // Isso é um comentário de uma linha
  $nome = "Aluno"; // Declara uma variável
  echo "Olá, " . $nome . "!"; // Imprime uma saída no navegador
?>

2. Primeiros Passos em PHP

Agora que já sabemos o que é o PHP, vamos colocar a mão na massa! Veremos como armazenar dados, tomar decisões e repetir ações.

Variáveis

Variáveis são "caixas" onde guardamos informações. No PHP, toda variável começa com o cifrão ($).

PHP 
<?php
  $texto = "Aprendendo PHP"; // Uma string (texto)
  $ano = 2023;              // Um número inteiro
  $preco = 19.99;           // Um número com casas decimais (float)
  $ehVerdade = true;        // Um booleano (verdadeiro ou falso)
?>

Operadores

Podemos fazer cálculos e comparações com os valores das variáveis.

  • Aritméticos: + (soma), - (subtração), * (multiplicação), / (divisão).
  • Comparação: == (igual), != (diferente), > (maior que), < (menor que).

Estruturas de Controle

Elas permitem que seu programa tome decisões e execute tarefas repetitivas.

Condicional if

Executa um bloco de código apenas se uma condição for verdadeira.

PHP 
<?php
  $idade = 18;
  if ($idade >= 18) {
    echo "Você é maior de idade.";
  } else {
    echo "Você é menor de idade.";
  }
?>

Laço de Repetição while

Executa um bloco de código enquanto uma condição for verdadeira.

PHP 
<?php
  $contador = 1;
  while ($contador <= 5) {
    echo "O número é: " . $contador . "<br>";
    $contador++; // Importante: incrementa o contador para não criar um loop infinito!
  }
?>

Laço de Repetição for

Ótimo para quando você sabe exatamente quantas vezes quer repetir algo.

PHP 
<?php
  for ($i = 0; $i < 5; $i++) {
    echo "Executando o loop pela vez número " . ($i + 1) . "<br>";
  }
?>

Funções

Funções são blocos de código reutilizáveis que executam uma tarefa específica. Você pode criar as suas!

PHP 
<?php
  // Definindo a função
  function saudacao($nome) {
    return "Olá, " . $nome . "! Seja bem-vindo(a).";
  }

  // Chamando a função
  echo saudacao("Maria");
  echo saudacao("João");
?>

3. Modularização: Gerenciamento de Arquivos

Um dos grandes poderes do PHP é a capacidade de interagir com o sistema de arquivos do servidor. Isso significa que seu script pode ler, escrever, copiar e deletar arquivos!

Abrindo e Fechando Arquivos

Para manipular um arquivo, primeiro você precisa abri-lo e informar o modo de abertura. No final, é uma boa prática fechá-lo.

  • 'r' - Leitura (apenas lê o arquivo).
  • 'w' - Escrita (apaga o conteúdo existente e escreve do zero).
  • 'a' - Adição (adiciona conteúdo ao final do arquivo).
PHP 
<?php
  $arquivo = 'meu_arquivo.txt';
  $conteudo = "Este é um novo conteúdo.\n";

  // Abre o arquivo para escrita ('w')
  $ponteiro = fopen($arquivo, 'w');

  // Escreve o conteúdo no arquivo
  fwrite($ponteiro, $conteudo);

  // Fecha o arquivo
  fclose($ponteiro);

  echo "Arquivo escrito com sucesso!";
?>

Lendo um Arquivo

Para ler o conteúdo, usamos a função fread ou fgets (para ler linha por linha).

PHP 
<?php
  $arquivo = 'meu_arquivo.txt';

  // Verifica se o arquivo existe antes de tentar ler
  if (file_exists($arquivo)) {
    // Lê o conteúdo inteiro do arquivo para uma string
    $conteudo = file_get_contents($arquivo);
    echo $conteudo;
  } else {
    echo "O arquivo não foi encontrado.";
  }
?>

4. Banco de Dados e POO

Aqui entramos em dois tópicos avançados e essenciais: como o PHP se comunica com bancos de dados e como podemos usar a Programação Orientada a Objetos (POO) para organizar melhor nossos projetos.

Banco de Dados

Um site dinâmico quase sempre precisa armazenar informações (usuários, produtos, posts, etc.). Para isso, usamos um Sistema Gerenciador de Banco de Dados (SGBD), como o MySQL.

O PHP se conecta ao MySQL, envia comandos em uma linguagem chamada SQL e recebe os dados para exibir na página.

SQL (Structured Query Language) é a linguagem padrão para falar com a maioria dos bancos de dados relacionais.

Conectando ao MySQL com PHP

A forma mais moderna e segura de se conectar é usando a extensão MySQLi ou PDO. Vemos um exemplo com MySQLi.

PHP 
<?php
  $servidor = "localhost";
  $usuario = "root";
  $senha = "";
  $nome_banco = "meu_site";

  // Criar a conexão
  $conn = new mysqli($servidor, $usuario, $senha, $nome_banco);

  // Verificar a conexão
  if ($conn->connect_error) {
    die("Falha na conexão: " . $conn->connect_error);
  }
  echo "Conectado com sucesso!";
?>

Programação Orientada a Objetos (POO)

POO é uma forma de programar que tenta "modelar" o mundo real. Em vez de pensarmos em variáveis e funções soltas, criamos objetos. Um objeto é uma instância de uma classe.

Uma classe é como um "molde" ou "planta". Ela define as propriedades (características) e os métodos (ações/comportamentos) que um objeto daquele tipo terá.

Exemplo de Classe em PHP

Vamos criar uma classe `Pessoa` com propriedades (nome, idade) e um método (se apresentar).

PHP 
<?php
  class Pessoa {
    // Propriedades
    public $nome;
    public $idade;

    // Método Construtor: executado quando o objeto é criado
    function __construct($nome, $idade) {
      $this->nome = $nome;
      $this->idade = $idade;
    }

    // Método
    public function seApresentar() {
      return "Olá, meu nome é " . $this->nome . " e eu tenho " . $this->idade . " anos.";
    }
  }

  // Criando um objeto (instância) da classe Pessoa
  $pessoa1 = new Pessoa("Carlos", 30);

  // Usando o método do objeto
  echo $pessoa1->seApresentar();
?>

A palavra-chave $this se refere ao próprio objeto em que o código está sendo executado. É como dizer "o nome deste objeto".

5. Recapitulando

Ufa! Vimos bastante coisa, não é mesmo? Vamos relembrar os pontos principais desta jornada:

  • PHP é uma linguagem de servidor para criar sites dinâmicos.
  • Usamos variáveis ($var) para armazenar dados.
  • Estruturas de controle como if, while e for dão lógica ao nosso código.
  • Funções nos ajudam a reutilizar código.
  • O PHP pode ler e escrever arquivos no servidor.
  • A conexão com bancos de dados (como MySQL) permite armazenar e recuperar informações de forma persistente.
  • A Programação Orientada a Objetos (POO) nos ajuda a organizar projetos complexos usando classes e objetos.
