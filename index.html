Slide 1: Introdução à Lógica de Programação

A lógica de programação é a base para resolver problemas computacionais de forma estruturada. É como criar uma receita detalhada para que o computador execute tarefas.

Passo 1: Entenda um algoritmo como uma sequência finita de ações executáveis.

Passo 2: Exemplos do dia a dia: Algoritmo para fazer café (passos: ferver água, adicionar pó, etc.).

Passo 3: Importância: Desenvolve raciocínio lógico e resolução de problemas.

Construa: Pense em um problema simples da sua rotina e liste os passos para resolvê-lo.

Slide 2: O que é Portugol?

Portugol é uma linguagem de pseudocódigo em português, usada para ensinar lógica sem focar em sintaxe complexa. É implementada em ferramentas como Visualg.

Passo 1: Vantagens: Fácil leitura, próxima da linguagem natural.

Passo 2: Diferenças de linguagens reais: Não compila para executável, mas simula execução.

Passo 3: Exemplo básico: Saudação.


algoritmo "Saudacao"
inicio
    escreval("Bem-vindo ao Portugol!")
fimalgoritmo

Construa: Compare com uma linguagem como Python: print("Bem-vindo").

Slide 3: Estrutura Básica de um Programa

Todo programa em Portugol inicia com "algoritmo 'nome'", declara variáveis com "var", executa no "inicio" e termina com "fimalgoritmo".

Passo 1: Cabeçalho: algoritmo "NomeDoPrograma"

Passo 2: Declaração de variáveis.

Passo 3: Bloco de execução: inicio ... fimalgoritmo


algoritmo "EstruturaBasica"
var
    mensagem: caracter
inicio
    mensagem <- "Olá!"
    escreval(mensagem)
fimalgoritmo

Construa: Adicione uma variável e exiba seu valor.

Slide 4: Variáveis e Tipos de Dados

Variáveis armazenam dados. Tipos: inteiro (números inteiros), real (decimais), caracter (textos), logico (verdadeiro/falso).

Passo 1: Declaração: var nome: caracter

Passo 2: Atribuição: nome <- "Ana"

Passo 3: Exemplo com múltiplas variáveis.


algoritmo "Variaveis"
var
    idade: inteiro
    altura: real
    aprovado: logico
inicio
    idade <- 20
    altura <- 1.75
    aprovado <- verdadeiro
    escreval("Idade: ", idade, " Altura: ", altura, " Aprovado: ", aprovado)
fimalgoritmo

Construa: Crie variáveis para nome, nota e status, e exiba.

Slide 5: Operadores Aritméticos

Usados para cálculos: + (soma), - (subtração), * (multiplicação), / (divisão), % (módulo), ^ (potência).

Passo 1: Soma simples: a + b

Passo 2: Exemplo com módulo: resto da divisão.

Passo 3: Potência: 2 ^ 3 = 8


algoritmo "Aritmeticos"
var
    a, b, resultado: inteiro
inicio
    a <- 10
    b <- 3
    resultado <- a % b  // Resultado: 1
    escreval("Resto: ", resultado)
fimalgoritmo

Construa: Calcule área de um retângulo (comprimento * largura).

Slide 6: Operadores Relacionais e Lógicos

Relacionais: >, <, >=, <=, =, <>. Lógicos: e (AND), ou (OR), nao (NOT).

Passo 1: Comparação: se (a > b)

Passo 2: Lógico: se (a > 0 e b < 10)

Passo 3: Negação: se (nao aprovado)


algoritmo "RelacionaisLogicos"
var
    nota: real
inicio
    nota <- 7.5
    se (nota >= 6 e nota <= 10) entao
        escreval("Aprovado")
    fimse
fimalgoritmo

Construa: Verifique se um número é par e positivo.

Slide 7: Entrada de Dados

Use "leia(variavel)" para obter input do usuário.

Passo 3: Exemplo com validação básica.

Slide 9: Estruturas Condicionais: Se-Senão

Decisões: se (condicao) entao ... senao ... fimse

Passo 1: Condição simples.

Passo 2: Adicionar else.

Passo 3: Exemplo com operadores lógicos.


algoritmo "SeSenao"
var
    temperatura: real
inicio
    leia(temperatura)
    se (temperatura > 30) entao
        escreval("Quente")
    senao se (temperatura < 10) entao
        escreval("Frio")
    senao
        escreval("Agradável")
    fimse
fimalgoritmo

Slide 10: Estruturas Condicionais: Se Aninhado

Condicionais dentro de outras para decisões complexas.

Passo 1: Se dentro de se.

Passo 2: Evite aninhamentos profundos para clareza.

Passo 3: Exemplo: Verificar login.


algoritmo "Aninhado"
var
    usuario, senha: caracter
inicio
    leia(usuario)
    leia(senha)
    se (usuario = "admin") entao
        se (senha = "123") entao
            escreval("Acesso concedido")
        senao
            escreval("Senha errada")
        fimse
    senao
        escreval("Usuário inválido")
    fimse
fimalgoritmo

Construa: Verifique se um ano é bissexto (divisível por 4, mas não por 100, exceto por 400).

Slide 11: Estruturas Condicionais: Escolha-Caso

Para múltiplas opções: escolha variavel caso valor ... outrocaso ... fimescolha

Passo 2: Casos específicos.

Passo 3: Caso default.

Slide 12: Laços de Repetição: Para

Repete um número fixo de vezes: para var de inicio ate fim [passo] faca ... fimpara

Passo 1: Contador simples.

Passo 2: Com passo.

Passo 3: Exemplo: Tabuada.


algoritmo "Para"
var
    i, num: inteiro
inicio
    leia(num)
    para i de 1 ate 10 faca
        escreval(num, " x ", i, " = ", num * i)
    fimpara
fimalgoritmo

Construa: Some números de 1 a 100.

Slide 13: Laços de Repetição: Enquanto

Repete enquanto condição for verdadeira: enquanto (condicao) faca ... fimenquanto

Passo 1: Verifique condição antes.

Passo 2: Evite loops infinitos.

Passo 3: Exemplo: Contagem regressiva.


algoritmo "Enquanto"
var
    contador: inteiro
inicio
    contador <- 10
    enquanto (contador > 0) faca
        escreval(contador)
        contador <- contador - 1
    fimenquanto
    escreval("Fim!")
fimalgoritmo

Construa: Peça números até digitar 0 e some-os.

Slide 14: Laços de Repetição: Repita-Até

Executa pelo menos uma vez: repita ... ate (condicao)

Passo 1: Bloco primeiro, condição depois.

Passo 2: Útil para menus.

Passo 3: Exemplo: Validação de input.


algoritmo "RepitaAte"
var
    opcao: caracter
inicio
    repita
        escreval("Digite S para sair: ")
        leia(opcao)
    ate (opcao = "S")
    escreval("Saiu!")
fimalgoritmo

Construa: Peça senha até acertar.

Slide 15: Vetores (Arrays Unidimensionais)

Armazenam múltiplos valores: var vetor[ tamanho ]: tipo

Passo 1: Declaração e inicialização.

Passo 2: Acesso: vetor[índice]

Passo 3: Exemplo: Média de notas.


algoritmo "Vetores"
var
    notas[5]: real
    i: inteiro
    soma: real
inicio
    soma <- 0
    para i de 0 ate 4 faca
        leia(notas[i])
        soma <- soma + notas[i]
    fimpara
    escreval("Média: ", soma / 5)
fimalgoritmo

Construa: Encontre o maior valor em um vetor.

Slide 16: Matrizes (Arrays Bidimensionais)

Slide 19: Exemplo Integrado: Jogo de Adivinhação

Combine conceitos: Gerar número aleatório, loops, condicionais.

Passo 1: Gerar número (use rand ou similar em Visualg).

Passo 2: Loop de tentativas.

Passo 3: Dicas com condicionais.


algoritmo "Adivinhacao"
var
    secreto, tentativa: inteiro
inicio
    secreto <- randi(100) + 1  // Em Visualg, use aleatorio
    repita
        leia(tentativa)
        se (tentativa > secreto) entao
            escreval("Muito alto!")
        senao se (tentativa < secreto) entao
            escreval("Muito baixo!")
        senao
            escreval("Acertou!")
        fimse
    ate (tentativa = secreto)
fimalgoritmo

Construa: Adicione contagem de tentativas e high score.
